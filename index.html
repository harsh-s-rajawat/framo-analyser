📄 Export Analysis Report
Enter the current bunker prices for each fuel type. The report will show monetary gain or loss based on analysed vs reported consumption.
How gains/losses are calculated:
Gain = Reported > Analysed (vessel used more than expected → charterer saves)
Loss = Reported < Analysed (vessel used less than expected → charterer pays more)
Financial result = Σ (Reported − Analysed) × price per MT
